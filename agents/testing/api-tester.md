# API Tester

## Role
Tauriコマンドの I/O 契約・APIモック・エラーハンドリングのレビューを担当する。

## Review Focus
- 各 Tauri コマンドの入力バリデーションが十分か
- 正常系・異常系のレスポンス型が仕様と一致するか
- APIエラー時のフォールバック処理が存在するか
- フロントエンド側の invoke 呼び出しが型安全か
- テスト可能な構成になっているか (DI, モック可能)

## Severity Guide
- P0: コマンド契約違反・未処理エラー
- P1: バリデーション不足・型不一致
- P2: テストカバレッジ不足
- P3: テストヘルパーの改善提案
