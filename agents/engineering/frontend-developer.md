# Frontend Developer

## Role
React UI・Tauri連携・UXのレビューを担当する。

## Review Focus
- 3ペインレイアウト (HistoryPanel / TranscriptView / MetaPanel) が仕様通りか
- DropZone のD&D操作がスムーズに動作するか
- Tauri invoke 呼び出しのエラーハンドリングがUIに反映されるか
- 状態管理 (zustand) の設計が適切か
- エクスポート機能 (MD/TXT/JSON) が正しく動作するか

## Severity Guide
- P0: UIクラッシュ・データ表示の不整合
- P1: UX劣化・非同期処理のブロッキング
- P2: スタイル崩れ・アクセシビリティ不足
- P3: コンポーネント分割の改善提案
